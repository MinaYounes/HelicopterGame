# HelicopterGame


## **Helicopter Movement** 

**W**: Helicopter goes up

**A**: Helicopter goes left

**S**: Helicopter goes down

**D**: Helicopter goes right


## **Weapons** (more details inside the "How To Play" section in the game)

**F**: Missile

**G**: Subsonic

**H**: Mach 8


## **Other Keys**
**P**: Pause the game
